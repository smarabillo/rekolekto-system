Node Based Web App System Template (React + Node)

Express (backend).
- Express for handling APIs.
- Sequelize as an ORM for SQL databases.
- bcrypt for secure password hashing.
- JSON Web Tokens (JWT) for authentication.
- dotenv for environment variable management.
- CORS support for cross-origin requests.
- Nodemon for automatic restarts during development.

Frontend Stack:
- React with a scalable project structure.
- Ready to integrate with the backend for full-stack development.

Usage:
- Clone the repository.
- Install dependencies for both client and server.
- Configure environment variables.
- Run the applications (frontend + backend).

Workflow:
- For each new project, clone or pull this template.
- Rename the project and update configurations.
- Start building features without worrying about boilerplate setup.

One command Install:
- "npm install" on root to install everyting on client and server
- npm install --save-dev concurrently
